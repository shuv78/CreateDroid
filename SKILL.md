# CreateDroid — Full-Stack Android App Factory

> **GitHub:** [shuv78/CreateDroid](https://github.com/shuv78/CreateDroid)  
> ⭐ **If this helps you, star the repo — it helps others find it too.**

**Build, fix, modify, and ship Android apps in minutes.** 37 app templates, 18 feature modules, 4 design systems, 2 platform stacks (Cordova+Firebase / Capacitor+Supabase), automated build-ship pipeline, and a self-learning bug database.

---

## 🔥 Why CreateDroid?

Tired of starting Android projects from scratch every time? CreateDroid is a **complete app factory** for Android mobile apps:

| You want… | CreateDroid gives you… |
|:----------|:-----------------------|
| A new app | 37 pre-built category templates — just pick your type |
| To fix a bug | Self-learning bug pattern database + 3-layer regression check |
| A feature | 18 feature modules ready to plug in (maps, payment, camera, scanner, chat, export…) |
| A different look | 4 design systems (LINE, Material You, Dark Minimal, iOS 19) |
| An APK delivered | One-command build → Telegram send pipeline |
| To learn from mistakes | Every fix is remembered — same bug never hits twice |
